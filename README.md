# Skills

Hermes Agent skills collection.

## Available Skills

| Skill | Description |
|-------|-------------|
| [kimi-cli](./skills/kimi-cli/) | Delegate coding tasks to Kimi CLI (Moonshot AI's coding agent) |
| [hermes-tweet](./skills/hermes-tweet/) | Use the Hermes Tweet plugin for tweet search, reply reads, X trends, and approval-gated X actions |

## Install

```bash
hermes skills install tmdgusya/skills/kimi-cli
hermes skills install tmdgusya/skills/hermes-tweet
```
